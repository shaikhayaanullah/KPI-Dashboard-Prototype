KPI Dashboard Prototype

Overview
A prototype Power BI dashboard built using dummy data to simulate FINANALYZE Recommendation Engine KPIs. Features financial health, investment behavior, recommendation performance, engagement analytics, and AUM tracking. Designed for future SQL Server integration and real-time business intelligence reporting.
This project is a prototype Power BI dashboard built using dummy data based on the FinAnalyz Recommendation Engine KPI requirements. The dashboard simulates a real-world financial analytics environment and is designed to support future integration with SQL Server data sources.

Purpose
The purpose of this project is to design and develop a prototype Power BI dashboard that demonstrates how financial, investment, engagement, recommendation, and AUM-related KPIs can be visualized and monitored using business intelligence tools. The project validates dashboard design, data modeling, KPI calculations, and reporting workflows before the availability of live SQL Server data.

Dashboard Modules
1. Financial Health Dashboard
* Monthly Surplus
* Savings Rate
* EMI Burden Ratio
* Idle Balance Analysis
* Income Stability Indicators

2. Investment Behavior Dashboard
* SIP Discipline Score

* Equity Exposure Analysis
* Risk Alignment Metrics
* Investor Segmentation

3. Recommendation Performance Dashboard
* Recommendation Conversion Rate

4. AUM & Revenue Dashboard
* Incremental AUM Tracking
* Quarterly AUM Growth
* Average Ticket Size
